Consider the following set of vectors: \(\langle 1, 1, 0 \rangle\), \(\langle 1, 1, 1 \rangle\) and \(\langle 0, 0, 1 \rangle\). Is it possible to find scalars \(x_1\), \(x_2\) and \(x_3\) so that \begin{gather*} x_1 \langle 1, 1, 0 \rangle + x_2 \langle 1, 1, 1 \rangle + x_3 \langle 0, 0, 1 \rangle = \langle 2, 3, 4 \rangle \end{gather*}
